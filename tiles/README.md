# 方块图片说明

将数字换成图片时，请把图片放在本目录下，并按**数字命名**：

| 数字 | 文件名   |
|------|----------|
| 2    | `2.png`  |
| 4    | `4.png`  |
| 8    | `8.png`  |
| 16   | `16.png` |
| 32   | `32.png` |
| 64   | `64.png` |
| 128  | `128.png`|
| 256  | `256.png`|
| 512  | `512.png`|
| 1024 | `1024.png`|
| 2048 | `2048.png`|
| 4096 | `4096.png`（可选）|
| 8192 | `8192.png`（可选）|

- 格式：PNG 或 JPG 均可（扩展名需与上面一致，如 `2.jpg` 则需在代码里改为 `.jpg`）。
- 尺寸：建议正方形，例如 200×200 像素，游戏会自动缩放填满方块。
- 若某数字没有对应图片，该格子会**自动显示数字**，不会报错。
